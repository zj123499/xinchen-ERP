📌 一、快速开始

1.1 登录系统

  1. 打开浏览器，访问系统地址（如 http://111.229.72.128:3000）
  2. 在登录页输入手机号（或用户名）和密码
  3. 首次登录时，系统会提示修改初始密码（默认密码：Xc@123456）
  4. 登录成功后自动跳转到工作台首页
⚠️ 安全提示：首次登录务必修改默认密码！密码至少 6 位。连续 5 次输错密码账号将被锁定。

1.2 界面导航

登录后主界面分为三个区域：

区域说明
左侧菜单栏所有功能模块入口。根据你的角色权限，会显示不同的菜单项
顶部导航栏显示当前用户姓名和角色，右上角为个人信息入口
主内容区当前页面的操作区和数据展示区

1.3 角色与可见菜单

不同角色看到的菜单不同。以下是典型角色的菜单对照：

角色主要菜单
超级管理员全部菜单（15 个一级模块）
市场专员工作台、销售管理、合同订单、客户回访、AI 智能
升学顾问工作台、交付管理、客户回访、AI 智能
财务工作台、财务管理（收款/成本/提成/薪资/报销/利润报表）
运营总监全部业务菜单 + 经营驾驶舱

📊 二、工作台（数据看板）

2.1 工作台概述

工作台是登录后的首页，展示与你相关的业务数据概览。根据你的角色权限，能看到的指标范围不同。例如：

  • 有 leads:view 权限 → 能看到线索统计数据
  • 有 payments:view 权限 → 能看到收款和成本数据
  • 没有任何业务 view 权限 → 只能看到员工数、合作方数等基本团队信息

2.2 核心指标卡片

指标含义使用场景
今日新增线索当天录入的新线索数量评估市场获客效果
待跟进线索状态为新线索/已联系的线索数追踪销售待办
本月签约本月已签订的合同数量衡量销售业绩
本月收款本月收款总额（含利润）监控现金流
本年业绩完成率当年收款 ÷ 当年合同额的百分比年度目标达成率

2.3 图表分析区

工作台下方提供多维度图表：

  • 线索分析：按来源（上门/转介绍/新媒体等）、按状态（新线索/已联系/已转化）、按负责人分布
  • 收款分析：按类型（客户服务费/学校返佣/合作方费用）、按月度分布
  • 合同分析：按月趋势、按业务线分布
  • 学生分析：按目标国家、按目标学位
  • 最近动态：最新 5 条线索和收款记录

📞 三、销售流程 — 从线索到签约

3.1 完整业务流程

录入线索→ 跟进沟通→ 转化为学生→ 创建合同→ 生成订单→ 服务交付

3.2 录入线索

  1. 点击左侧菜单「销售管理 → 线索管理」
  2. 点击右上角 + 新增线索 按钮
  3. 填写必填字段：姓名、手机号（必填且唯一，重复手机号会被拦截）
  4. 选填：微信、来源渠道、意向国家、意向专业、备注等
  5. 选择负责人（默认自己，主管可分配给他人）
  6. 点击保存
💡 撞单处理：如果录入的手机号已被其他顾问录入，系统会自动提示，并引导发起转介绍申诉流程。

3.3 线索跟进

  1. 在线索列表找到需要跟进的线索
  2. 点击线索姓名进入详情页
  3. 在「跟进记录」区域添加新的跟进：选择跟进方式（电话/微信/面谈/邮件），填写沟通内容
  4. 设置「下次联系时间」，系统会在到期时提醒
  5. 跟进后更新线索状态：新线索 → 已联系 → 已筛选 → 已转化

3.4 线索转化学生

  1. 当沟通深入、客户意向明确时，将线索转化为学生
  2. 在线索详情页点击「转学生」按钮
  3. 系统自动带入线索信息（姓名、手机号等）
  4. 补充详细学术信息：当前学历、GPA、语言成绩（雅思/托福/GRE/GMAT）、目标国家/专业/学位
  5. 保存后，该学生在「学生档案」中可查

3.5 线索搜索与筛选

线索列表页支持多种筛选方式：

  • 关键词搜索：按姓名、手机号、微信匹配
  • 状态筛选：新线索 / 已联系 / 已筛选 / 已转化 / 已无效
  • 来源筛选：上门、转介绍、新媒体、搜索引擎等

📝 四、合同与订单

4.1 创建合同

  1. 进入「合同订单 → 合同管理」
  2. 点击「新增合同」，选择关联学生
  3. 填写合同信息：合同金额、签约日期、有效期
  4. 绑定服务角色：签约顾问、规划顾问、文书顾问（可同一人）
  5. 选择业务线（留学申请 / 语言培训 / 背景提升等）
  6. 提交后，合同进入「待审批」状态
⚠️ 注意：如果合同优惠折扣超过 20%，需要走审批流（销售→主管→经理逐级审批）。

4.2 订单管理

合同签订后生成订单，订单支持分期付款。操作流程：

  1. 进入「合同订单 → 订单管理」
  2. 合同签约后自动生成订单，状态为「待付款」
  3. 录入首付款后，订单状态变为「进行中」
  4. 全部款项到账后，订单状态变为「已完成」

📬 五、交付管理 — 申请到入学

5.1 完整交付流程

创建申请→ 准备材料→ 递交申请→ 等待Offer→ 办理签证→ 入学注册

5.2 创建申请

  1. 进入「交付管理 → 申请管理」
  2. 点击「添加申请」，选择对应的学生
  3. 填写申请信息：目标院校、专业、学位、入学季节
  4. 关联学生材料清单（成绩单、推荐信、语言成绩等）
  5. 保存后，申请状态为「材料准备中」

5.3 材料管理

在申请详情页可管理材料清单：

  • 标记每项材料的上传状态（待提交 / 已提交 / 已审核）
  • 系统根据院校要求自动生成材料清单
  • 截止日期临近时自动提醒

5.4 Offer 与签证管理

  1. Offer 管理：收到 Offer 后在系统中录入（条件/无条件），标记接受或拒绝
  2. 签证管理：在签证模块创建签证记录，跟踪递交→面签→出签全流程
  3. 入学确认：学生入学后，订单状态标识为「已完成」

5.5 文书工作台

文书顾问的专属工作台：

  • 查看所有分配给自己的文书任务
  • 按学生查看 PS / CV / 推荐信的编写进度
  • 使用 AI 文书助手 辅助生成初稿和优化建议

💰 六、财务管理

6.1 收款录入

  1. 进入「财务管理 → 收款管理」
  2. 点击「添加收款」，选择关联学生
  3. 填写收款金额、收款类型（客户服务费/学校返佣/合作方费用/其他）
  4. 选择付款方式（银行转账/微信/支付宝/现金等）
  5. 财务年度和月份自动匹配当前时间
  6. 保存后，总金额自动反映在工作台和利润报表中
💡 多币种支持：系统支持 CNY/USD/GBP/AUD 等多币种，录入时选择币种即可自动换算。

6.2 成本管理

  1. 进入「财务管理 → 成本管理」
  2. 成本类型包括：市场费用、员工薪资、渠道佣金、运营开支等
  3. 每笔成本关联到财务年度和月份
  4. 成本数据影响利润报表的计算（利润 = 收款 - 成本）

6.3 提成与薪资

功能说明
提成规则配置销售提成比例，支持按里程碑分阶段释放（首款/尾款/Offer/入学）
提成明细自动计算每个顾问的提成金额，按签约合同和到账情况生成
薪资管理员工底薪 + 提成汇总 + 扣款，每月生成薪资单
报销管理员工提交费用报销，通过审批流审批后确认支付

6.4 财务报表

  • 利润报表：按月度统计收入/成本/利润
  • 应收应付：跟踪客户欠款和供应商应付款
  • 发票管理：开具和登记发票
  • 退费管理：退费申请→审批→执行退款→自动计算扣回提成

👥 七、员工管理

7.1 新增员工（自动建号）

  1. 管理员进入「系统设置 → 员工信息」
  2. 点击「新增员工」
  3. 填写姓名、手机号（必填，将作为登录账号）、性别、邮箱等
  4. 选择角色（可多选）
  5. 登录账号栏留空即可，系统自动用手机号创建账号
  6. 默认密码为 Xc@123456，首次登录强制修改
  7. 保存后员工即可用手机号登录
✅ 自动化：填写手机号 → 保存 → 账号自动创建完毕，无需单独操作。

7.2 员工离职处理

  1. 在员工列表找到要处理的员工
  2. 编辑员工，将状态改为「离职」
  3. 系统自动停用该员工的登录权限
  4. 其名下的线索/学生/订单等数据不会丢失
  5. 管理员在列表点击该员工的「重新分配数据」按钮
  6. 选择目标接收人，一键将所有业务数据转移

7.3 批量同步账号

如果老员工的登录账号还是工号（如 EMP2025001），可使用同步账号功能一键改为手机号：

  1. 在员工列表页工具栏点击「同步账号」
  2. 系统自动将所有员工的登录账号更新为手机号
  3. 已完成同步的员工不会被重复处理

7.4 角色权限配置

  1. 在员工列表页工具栏点击「配置权限」
  2. 系统自动为每个角色按业务语义分配 view 权限
  3. 也可在「系统设置 → 角色权限」中手动调整
  4. 权限生效：用户重新登录后即可看到对应菜单和数据

⚙️ 八、系统设置

数据字典

管理系统中的下拉选项：线索来源、学生状态、申请状态、合同类型等。管理员可自定义。

系统配置

全局参数设置：公司名称、AI 网关地址/密钥、钉钉集成配置等。

审计日志

记录所有关键数据的变更：谁、什么时间、改了哪个字段、改前/改后的值。不可删除。

操作日志

记录用户的操作行为：登录、查看、导出、删除等。

钉钉集成

支持钉钉 Webhook：入职自动建账号、离职自动停用、事件同步。

审批流配置

配置业务审批流程：合同优惠、退费审批、报销审批、线索划转。支持多级、会签、或签。

每天必做清单

  • ✅ 查看工作台：了解今日新线索和待跟进线索
  • ✅ 跟进你的线索：及时更新跟进记录和线索状态
  • ✅ 处理待审批事项：进入审批中心处理待审
  • ✅ 检查交付进度：关注申请和签证的到期时间

数据安全须知

  • 不要将登录账号借给他人使用
  • 离开工位时关闭浏览器或锁屏
  • 1 小时内导出超过 50 条数据会触发安全锁定
  • 所有数据操作均记录在审计日志中，不可删除
