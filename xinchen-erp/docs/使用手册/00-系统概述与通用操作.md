# 新辰留学 ERP — 系统使用手册（总览）

> 适用版本：2026-07-18 线上部署版（腾讯云 111.229.72.128:3000）

## 一、系统简介

新辰留学 ERP 是面向留学服务公司的全流程业务管理系统，覆盖「线索 → 学生 → 合同 → 订单 → 申请 → Offer → 签证 → 入学 → 转介绍」的学生全生命周期，并集成财务利润、客户成功、风险管理与 AI 智能助手。

- **访问地址**：`http://111.229.72.128:3000`（正式域名 `erp.xinchen.com` 待配置）
- **部署形态**：Docker 容器化，已上线运行

## 二、登录

1. 浏览器访问系统地址，进入登录页。
2. 输入「账号 / 密码」，点击 **登录**。
3. 登录成功进入「工作台（Dashboard）」。
4. 右上角头像可「退出登录」；登录态失效后需重新登录。

**当前可用账号**：

| 角色 | 账号 | 密码 | 说明 |
|------|------|------|------|
| 超级管理员 | `admin` | `admin123` | 系统唯一启用角色，拥有全部权限 |

> 说明：销售顾问 / 交付顾问 / 财务 / 部门主管 四个角色当前未在系统中配置，其对应演示账号（sales01、sales02、delivery01、finance01）已删除。如需启用这些岗位，请在「系统设置 → 角色权限」中重新创建角色并分配菜单 / 权限。初始密码为演示密码，正式上线后请修改。

## 三、界面布局

- **左侧**：功能导航栏（按角色权限显示不同菜单）。
- **顶部**：全局搜索、通知中心、AI 助手入口、用户头像。
- **中间**：当前模块工作区（列表 / 表单 / 看板）。

## 四、通用操作

- **列表页**：支持筛选、搜索、分页；部分列表支持「导出」（受按钮权限控制）。
- **表单页**：必填项标红；保存前请确认信息完整。
- **通知中心**：系统消息、风险预警、审批待办会推送到通知中心，并同步钉钉。
- **数据权限**：普通员工仅见「本人」数据；主管可见「本部门」；管理员可见「全部」。

## 五、权限模型（四层）

1. **菜单权限**：决定能看到哪些菜单 / 页面。
2. **按钮权限**：决定能否 新增 / 编辑 / 删除 / 导出 / 审批。
3. **数据权限**：本人 / 本部门 / 全部。
4. **字段组权限**：按字段组（如财务块、联系方式块）控制 可读 / 可写。
