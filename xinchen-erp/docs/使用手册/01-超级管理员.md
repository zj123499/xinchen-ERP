# 超级管理员使用手册

## 角色定位

系统最高权限，负责系统初始化、组织架构搭建、角色与权限配置、基础数据维护，并拥有全量业务数据的查看与管理能力，是系统的「总开关」。

## 可见菜单

工作台、销售管理（线索 / 学生 / 跟进）、合同订单（合同 / 订单）、交付管理（申请 / Offer / 签证）、客户回访、财务管理（全部）、扩展业务（租房 / 境外 / 合作方）、营销管理（站群 / 新媒体）、系统设置（组织 / 角色 / 字典 / 配置）、数据大屏。

## 核心工作

### 1. 组织架构搭建（系统设置 → 组织架构）
- 维护部门（部门树）、岗位、员工，并将员工与系统账号、部门关联。
- 已对接钉钉，组织数据可由钉钉事件同步（departments / positions / employees / user_departments）。

### 2. 角色与权限（系统设置 → 角色权限）
- 内置角色：超级管理员 / 销售顾问 / 交付顾问 / 财务 / 部门主管。
- 为角色勾选「菜单权限」与「按钮权限」；为部门主管等配置「数据权限 = 本部门」。
- 字段组权限：控制角色能否查看 / 编辑敏感字段组（如学生财务块、合同财务条款）。

### 3. 数据字典（系统设置 → 数据字典）
- 维护意向等级、学历、目标国家、线索来源等枚举值，供各模块下拉选择。

### 4. 系统配置（系统设置 → 系统配置）
- 公司名称、默认币种、财年起止、回访间隔。
- 「AI 智能配置」：填写 AI 网关地址 / 密钥 / 模型（无 Key 时自动降级为模板）。

### 5. 全量业务监管
- 可进入任意业务模块查看 / 操作全量数据。
- **数据大屏**（`/bi/screen`）：实时展示学生申请进度 5 节点（材料准备 → 递交申请 → 递交签证 → 到校注册 → 结案）的漏斗与滚动进度墙。
- **经营驾驶舱**：今日概览、销售漏斗、利润、人效、风险、渠道 ROI。

### 6. 审批流初始化（如未启用）
- 在审批中心可「一键初始化默认审批流」，创建 退费 / 报销 / 合同优惠 / 线索划转 四类审批流（含会签 / 或签）。

## 注意事项

- 角色 / 权限修改即时生效，请谨慎授予「删除」「导出」「审批」等高危按钮。
- 关键配置（提成规则、审批流、表单 Schema）改动会自动保留**版本历史**，可在配置版本管理中回溯。
- 所有关键操作均记录到**审计日志**与**操作日志**，请勿进行与业务无关的操作。
- 离职盘点：员工离职前，可将其名下未签约线索转入交接池交接，阻断私下转移客户。
